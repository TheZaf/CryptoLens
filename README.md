## 🚀 Crypto Monitoring Web App

A full-stack cryptocurrency tracking platform that provides real-time price insights and interactive data visualizations. Built with a modern tech stack, this app delivers a seamless and secure experience for users who want to explore detailed crypto analytics.

### ✨ Features

* 📊 **Interactive Charts**

  * Built with Recharts for responsive, dynamic data visualization
  * View historical price trends with smooth and customizable charts
  * Clean UI optimized for readability and performance

* 🔐 **Authentication System**

  * Secure user authentication using JSON Web Tokens (JWT)
  * Features include:

    * Sign up
    * Login
    * Logout
  * Protected routes: only authenticated users can access detailed crypto charts

* 💰 **Real-Time Crypto Data**

  * Integrated with the CoinGecko API
  * Fetches up-to-date cryptocurrency prices, trends, and market data

* ⚡ **State Management**

  * Lightweight and efficient global state handling with Zustand

* 🌐 **API & Backend**

  * RESTful API built with Express.js
  * Handles authentication, user sessions, and secure data flow

* 🎨 **Modern UI**

  * Styled using Tailwind CSS for a clean, responsive design
  * Mobile-friendly and optimized for all screen sizes

* 🔄 **Efficient Data Fetching**

  * Axios used for API communication with proper error handling and performance optimization

---

### 🛠️ Tech Stack

**Frontend:**

* React.js
* Tailwind CSS
* Recharts
* Zustand
* Axios

**Backend:**

* Node.js
* Express.js
* JSON Web Token (JWT)

**API:**

* CoinGecko API
